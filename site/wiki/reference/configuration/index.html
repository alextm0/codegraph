    Configuration Reference

    All CodeGraph configuration lives in config.yaml at the project root. The file is loaded once at startup and passed through the system via dependency injection.

    Full Configuration File

    # config.yaml

# ── Project Settings ──────────────────────────────────────────
project_root: "."                # Root of the codebase to index

exclude_patterns:                # Paths/patterns to skip during parsing
  - "__pycache__"
  - ".venv"
  - ".git"
  - "*.pyc"
  - ".pytest_cache"

# ── Neo4j Connection ──────────────────────────────────────────
neo4j:
  uri: "neo4j://localhost:7687"  # Connection URI
  username: "neo4j"              # Database username
  password: "your_password"      # Database password — keep this secret
  database: "neo4j"              # Database name (use "neo4j" for the default)

# ── Personalized PageRank ─────────────────────────────────────
ppr:
  damping_factor: 0.85     # 1 − alpha; higher = walks spread further from seeds
  max_iterations: 20       # Convergence iterations (20 is sufficient)
  tolerance: 1.0e-7        # Early stopping threshold
  top_k: 20                # How many PPR results to return before post-processing

# ── Seed Selection ────────────────────────────────────────────
seed_selection:
  entity_match_weight: 0.6  # Weight for explicitly mentioned entities
  bm25_weight: 0.3          # Weight for BM25 keyword search results
  current_file_weight: 0.1  # Weight for entities in the current file
  bm25_top_n: 5             # How many BM25 top results to include as seeds

# ── MCP Server ────────────────────────────────────────────────
mcp:
  server_name: "codegraph"    # Server name shown to MCP clients
  default_token_budget: 6000  # Max tokens in the response to the AI agent
  default_top_k: 15           # Default number of results returned by get_relevant_context

    Field Reference

    Project Settings

    Field Type Default Description
    project_root string "." Root directory of the codebase to parse. Relative paths are resolved from the working directory.
    exclude_patterns list[string] See above Files/directories matching these patterns are skipped during parsing. Supports glob syntax.

    Neo4j Connection

    Field Type Default Description
    uri string "neo4j://localhost:7687" Neo4j connection URI. Use bolt:// for local non-TLS, neo4j+s:// for TLS-encrypted cloud instances.
    username string "neo4j" Database username.
    password string — Database password. Never commit to version control.
    database string "neo4j" Name of the database to use. The default Neo4j installation uses "neo4j".

    Connection URIs by environment:

    Environment URI Format
    Local (Neo4j Desktop) neo4j://localhost:7687
    Docker neo4j://localhost:7687
    Neo4j AuraDB (cloud) neo4j+s://xxxxxxxx.databases.neo4j.io

    PPR Settings

    Field Type Default Description
    damping_factor float (0–1) 0.85 Probability of continuing the random walk (= 1 − restart probability). Higher values spread further from seeds. 0.85 is the standard PageRank default.
    max_iterations int 20 Maximum convergence iterations. Scores typically converge by iteration 15–20.
    tolerance float 1e-7 Early stopping: if max score change between iterations is below this, stop early.
    top_k int 20 How many nodes PPR returns (by score) before token budgeting. Setting higher means post-processing sees more candidates.

    Effect of damping_factor: - 0.95 → walk spreads very far from seeds; results include loosely related code - 0.85 → balanced (recommended) - 0.70 → walk stays close to seeds; results tightly clustered around mentioned entities

    Seed Selection Settings

    Field Type Default Description
    entity_match_weight float (0–1) 0.6 Weight given to entities explicitly mentioned by the AI agent.
    bm25_weight float (0–1) 0.3 Weight given to entities found via BM25 text search of the task description.
    current_file_weight float (0–1) 0.1 Weight given to entities in the current file (if provided).
    bm25_top_n int 5 How many top BM25 results to include in the seed set.

    Constraint: entity_match_weight + bm25_weight + current_file_weight should equal 1.0. The system normalizes the final personalization vector, so the exact sum doesn't matter, but it should be close.

    MCP Server Settings

    Field Type Default Description
    server_name string "codegraph" Name shown to MCP clients in their tool list.
    default_token_budget int 6000 Maximum tokens in the context returned by get_relevant_context. Can be overridden per-call.
    default_top_k int 15 Default number of results returned. Can be overridden by the AI agent when calling the tool.

    Loading Configuration in Code

    from src.graph.connection import load_full_config

config = load_full_config('config.yaml')

# Access sections
neo4j_config = config['neo4j']
ppr_config = config['ppr']
seed_config = config['seed_selection']
mcp_config = config['mcp']

    The load_full_config() function reads the YAML file and returns a plain dict. Configuration is passed explicitly as function arguments — no global state.
