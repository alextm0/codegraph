    Installation

    This page covers installing CodeGraph's Python dependencies and configuring it to connect to Neo4j.

    1. Clone the Repository

    git clone <repo-url>
cd codegraph

    2. Install Dependencies

    Install the package in editable mode with all development dependencies:

    python -m pip install -e ".[dev]"

    The [dev] extras include pytest and other testing tools. The -e flag installs in editable mode so your source changes are reflected immediately.

    Verify the install worked: 

    python -c "import tree_sitter; import neo4j; import mcp; print('OK')"

    3. Configure Neo4j Connection

    Open config.yaml in the project root and update the neo4j section:

    neo4j:
  uri: "neo4j://localhost:7687"
  username: "neo4j"
  password: "your_password_here"   # ← change this
  database: "neo4j"

    Important: Never commit your real password to version control. The config.yaml is gitignored by default.

    4. Verify the Connection

    Run the connection check:

    python -c "
from src.graph.connection import create_driver, verify_connectivity, load_full_config
config = load_full_config('config.yaml')
driver = create_driver(config['neo4j'])
verify_connectivity(driver)
print('Connected to Neo4j successfully')
driver.close()
"

    If this prints Connected to Neo4j successfully, you're ready to go.

    5. Run the Tests

    Confirm everything works end-to-end:

    # Run all tests (requires Neo4j running)
python -m pytest tests/ -v

# Run only tests that don't need Neo4j
python -m pytest tests/test_graph_resolution.py tests/test_seed_selection.py tests/test_post_processing.py -v

    Tests that require Neo4j are marked with @neo4j_required and skip gracefully if Neo4j isn't running.

    Configuration Reference

    The full config.yaml with all default values:

    project_root: "."
exclude_patterns:
  - "__pycache__"
  - ".venv"
  - ".git"
  - "*.pyc"
  - ".pytest_cache"

neo4j:
  uri: "neo4j://localhost:7687"
  username: "neo4j"
  password: "your_password"
  database: "neo4j"

ppr:
  damping_factor: 0.85       # 1 - alpha; higher = walks stay closer to seed
  max_iterations: 20
  tolerance: 1.0e-7
  top_k: 20                  # How many results PPR returns before post-processing

seed_selection:
  entity_match_weight: 0.6   # Weight for explicitly mentioned entities
  bm25_weight: 0.3           # Weight for BM25 keyword matches
  current_file_weight: 0.1   # Weight for entities in the current file
  bm25_top_n: 5              # How many BM25 results to include as seeds

mcp:
  server_name: "codegraph"
  default_token_budget: 6000  # Max tokens in MCP response
  default_top_k: 15           # Default number of results returned

    See Configuration Reference → for detailed explanations of each field.
